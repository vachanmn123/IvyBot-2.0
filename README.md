# IvyBot 2.0

---

**IvyBot 2.0** is a rewrite of the original IvyBot a general purpose discord bot with cool features.

---

## To run the bot:

1. clone the repository
   `git clone https://github.com/vachanmn123/IvyBot-2.0`
2. install the dependencies
   `npm install`
3. run the bot
   `node .`
